Kesimpulan Konsep
Mean sensitif terhadap outlier — satu nilai yang sangat besar atau kecil dapat menarik Mean jauh dari nilai-nilai lainnya.
Median tahan terhadap outlier — karena Median hanya melihat posisi tengah setelah data diurutkan, nilainya tidak berubah drastis.
Mean (Rata-rata)
Jumlah semua nilai data dibagi banyaknya data. Paling umum digunakan, tapi sensitif terhadap nilai ekstrem (outlier).
x̄ = (x₁ + x₂ + ... + xₙ) / n = Σxᵢ / n
✓ Menggunakan semua data ✓ Mudah dihitung ✗ Terpengaruh outlier 📌 Cocok: data homogen tanpa outlier
Median (Nilai Tengah)
Nilai yang berada di tengah setelah data diurutkan. Tidak terpengaruh outlier.
n ganjil: Me = x[(n+1)/2] n genap: Me = (x[n/2] + x[n/2+1]) / 2
✓ Tahan outlier ✓ Representatif untuk data miring ✗ Harus diurutkan dulu 📌 Cocok: data dengan outlier (gaji, harga properti)
Modus (Nilai Terbanyak)
Nilai yang paling sering muncul. Bisa lebih dari satu atau tidak ada.
Mo = nilai dengan frekuensi tertinggi
✓ Cocok untuk data kategorik ✗ Bisa tidak ada / lebih dari satu 📌 Cocok: ukuran baju/sepatu terlaris
